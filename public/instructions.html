Getting Started

Generate, sign & send documents directly from any monday.com board.

Prerequisites

  • A monday.com account (any plan)
  • At least one board with items to test document generation
  • A .docx template file (or use our built-in designer)
1

Add DocuGen Pro to your board

Open any board → click Views → select DocuGen Pro. The app opens as a new board view.

2

Upload or design a template

Upload a .docx file or click ✏️ Design to use the built-in editor. Use {{column_name}} placeholders to pull live data from your board columns.

3

Select an item and generate

Pick any board item → click ⬇ Word, ⬇ PDF, or ⬇ Excel. Your document is generated instantly with that item's data.

4

Send for e-signature (optional)

Go to the ✍️ Signatures tab → enter the signer's name and email → click Generate signature link. The signer receives a secure link and you get notified when they sign.

5

Automate with triggers

In the 🤖 Automation tab, set a column value trigger (e.g. Status = "Approved") and DocuGen Pro will auto-generate documents without manual intervention.
